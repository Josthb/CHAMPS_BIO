# CHAMPS_BIO
Ce projet réalisé en équipe de trois durant le premier semestre 2022-2023 met en pratique le HTML, le CSS ainsi que le PHP avec la méthode MVC. Ce projet nous a amené à créer un site pour une AMAP BIO. On y retrouve des informations sur cette dernière et ce qu'elle propose (abonnement à des paniers, achat de produits...).
